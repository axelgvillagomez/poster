Proyecto Final: Realidad Aumentada (NFT)

Estudiante: Axel García Villagómez

Tema: Reforestación Urbana (Aspecto Social/Ecológico)

1. Descripción del Proyecto

El proyecto consiste en un póster interactivo diseñado para promover la reforestación urbana. Utiliza tecnología de Realidad Aumentada basada en marcadores de Seguimiento de Características Naturales (NFT), permitiendo que cualquier smartphone con navegador web pueda visualizar contenido 3D dinámico al apuntar al póster.

2. Imagen del Póster Final

Póster final con código QR integrado para acceso directo.

3. Proceso Técnico

A. Diseño y Generación de Marcadores

Se diseñó un póster de alto contraste con elementos visuales únicos para facilitar el seguimiento (tracking). La imagen fue procesada utilizando la herramienta NFT-Marker-Creator para generar los conjuntos de datos necesarios (.fset, .fset3, .iset).

B. Desarrollo de la App AR

La aplicación utiliza ARnft como motor de seguimiento y Three.js para el renderizado 3D. Se implementó una escena que incluye:

  • Un modelo de árbol 3D procedimental (tronco y capas de follaje).
  • Animación de partículas (hojas flotantes) y balanceo suave.
  • Una interfaz de usuario (UI) que indica el estado del marcador y proporciona instrucciones.

C. Implementación de QR y Deployment

Se generó un código QR dinámico que apunta a la URL de hosting. El proyecto se encuentra desplegado en GitHub Pages para su acceso público instantáneo.

4. Tecnologías Utilizadas

  • HTML5 / CSS3 / JavaScript (ES6+)
  • Three.js: Renderizado de gráficos 3D.
  • ARnft: Seguimiento NFT (Natural Feature Tracking).
  • Puppeteer: Exportación y automatización de activos.
  • GitHub Pages: Hosting y despliegue continuo.
